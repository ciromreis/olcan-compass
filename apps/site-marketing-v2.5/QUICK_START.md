## 🚀 Installation

```bash
# Navigate to project
cd /Users/ciromoraes/Documents/THE-Code-Base/olcan-compass/apps/site-marketing-v2.5

# Install dependencies
npm install

# Copy environment template
cp .env.example .env.local

# Start development server
npm run dev
```

**Open**: http://localhost:3001
