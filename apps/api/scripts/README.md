# API Scripts

Operational and debugging scripts for `apps/api`.

## Structure

- `apps/api/scripts/seed/` - data seeding utilities
- `apps/api/scripts/admin/` - admin/user maintenance utilities
- `apps/api/scripts/debug/` - mapper/import debugging utilities

## Notes

- These scripts are not part of normal runtime.
- Run them from `apps/api` so module imports resolve correctly.
