## Leitura rápida

Hoje o Olcan Compass deve ser entendido como um micro-SaaS para internacionalização profissional com quatro frentes centrais:

1. diagnóstico e leitura de perfil
2. rotas executáveis
3. forge / dossiê
4. treino de entrevista

Marketplace entra como destravamento humano premium.

A camada hoje chamada de `companion` deve funcionar como leitura personalizada, retenção e atmosfera de produto. Ela não deve engolir a proposta principal.

## Importante

Documentos neste diretório que falem em produto “100% completo”, foco central em guildas, batalhas ou estúdio de conteúdo devem ser tratados como material histórico, não como direção ativa de implementação.
