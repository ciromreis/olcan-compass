# Guia de Testes Humanos (Validation)

## 🧠 Contexto BMAD
Os testes humanos capturam o "Sentimento do Breakthrough". No BMAD, a métrica final não é apenas o código compilar, mas o usuário sentir que seu problema foi resolvido de forma premium. Este guia orienta como conduzir essas sessões de validação para refinar a "Aura" do produto.

## Conteúdo

### Fluxos para Teste
1. **Onboarding**: O quiz OIOS é compreensível? O tempo de resposta é aceitável?
2. **First Value (Forge)**: O polimento de IA traz um "Aha Moment"?
3. **Navegação**: O HUD do companheiro Aura obstrui a visão ou é útil?

### Coleta de Dados
- **Feedback Qualitativo**: Gravações de tela e "Think Aloud".
- **Checklist de Bugs Visuais**: Desalinhamentos, cores fora do token, erros de gramática no copy.
