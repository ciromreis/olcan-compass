## Technical Stack

### Backend
- **Framework**: FastAPI
- **Database**: PostgreSQL (Neon) / SQLite (dev)
- **ORM**: SQLAlchemy 2.0 (async)
- **AI**: Google Gemini Flash
- **Payments**: Stripe
- **Auth**: JWT tokens

### Frontend
- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **State**: Zustand
- **Styling**: TailwindCSS
- **Animations**: Framer Motion
- **Editor**: Tiptap
- **Forms**: React Hook Form
- **Validation**: Zod

### Infrastructure
- **Hosting**: Vercel (frontend + API)
- **Database**: Neon Serverless Postgres
- **Storage**: Vercel Blob / S3
- **Monitoring**: Sentry + PostHog
- **Analytics**: PostHog

## Development Principles

1. **Feature-First**: Build complete features, not half-features
2. **Test Early**: Test each feature before moving on
3. **Document Always**: Update docs as you build
4. **User-Centered**: Every feature solves a real problem
5. **Performance Matters**: < 3s AI responses, 60fps animations
6. **Accessibility**: WCAG AA compliance from start
7. **Mobile-First**: Responsive design always
8. **Security**: Never compromise on auth/payments
