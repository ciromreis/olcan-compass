# Design System: OIOS Archetype Visual Integration

> **Document:** OIOS Visual Language
> **Version:** 1.0.0 | **Last Updated:** March 2026

## OIOS Visual Philosophy

The OIOS (Olcan Internationalization Operating System) is not just a categorization system—it's a visual language that adapts the entire interface to the user's psychological archetype and journey stage.

### Core Principles

**Dynamic Theming**
- Interface adapts to user's dominant archetype
- Color, typography, and motion reflect psychological state
- Evolution stages trigger visual transformations
- Fear clusters influence UI tone and guidance

**Metamodern Storytelling**
- Serious personal growth meets game-like progression
- Professional tools with whimsical archetype spirits
- Data-driven insights with emotional resonance
- Structured frameworks with fluid expression

## Dynamic UI Theming System

### Archetype Theme Application

**JavaScript Theme Manager:**
```typescript
class ArchetypeThemeManager {
  private currentArchetype: OIOSArchetype;
  private evolutionStage: EvolutionStage;
  
  applyArchetypeTheme(archetype: OIOSArchetype, stage: EvolutionStage) {
    const root = document.documentElement;
    
    // Set archetype color
    root.style.setProperty('--archetype-color', this.getArchetypeColor(archetype));
    
    // Set evolution stage properties
    root.style.setProperty('--stage-color-opacity', this.getStageOpacity(stage));
    root.style.setProperty('--stage-icon-size', this.getStageIconSize(stage));
    root.style.setProperty('--stage-glow-intensity', this.getStageGlow(stage));
    
    // Apply archetype class to body
    document.body.className = `archetype-${archetype} stage-${stage}`;
  }
  
  getArchetypeColor(archetype: OIOSArchetype): string {
    const colorMap = {
      institutional_escapee: '#2563EB',
      scholarship_cartographer: '#D4AF37',
      career_pivot: '#F59E0B',
      global_nomad: '#0EA5E9',
      technical_bridge_builder: '#059669',
      insecure_corporate_dev: '#EC4899',
      exhausted_solo_mother: '#10B981',
      trapped_public_servant: '#7C3AED',
      academic_hermit: '#8B5CF6',
      executive_refugee: '#64748B',
      creative_visionary: '#DB2777',
      lifestyle_optimizer: '#94A3B8'
    };
    return colorMap[archetype];
  }
  
  getStageOpacity(stage: EvolutionStage): string {
    return stage === 'rookie' ? '0.4' : stage === 'champion' ? '1' : '1';
  }
  
  getStageIconSize(stage: EvolutionStage): string {
    return stage === 'rookie' ? '32px' : stage === 'champion' ? '48px' : '64px';
  }
  
  getStageGlow(stage: EvolutionStage): string {
    return stage === 'rookie' ? '0.15' : stage === 'champion' ? '0.3' : '0.5';
  }
}
```
