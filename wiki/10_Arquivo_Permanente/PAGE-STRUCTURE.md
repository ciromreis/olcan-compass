## 🔄 Jornada do Usuário

```
1. ATENÇÃO (Hero)
   ↓
   "Você entrega resultado. Agora vão enxergar."
   
2. INTERESSE (Solução + Pain Points)
   ↓
   Identificação com as dores + Apresentação da solução
   
3. DESEJO (Timeline + Prova Social)
   ↓
   Como funciona + Resultados reais de outros clientes
   
4. AÇÃO (Oferta + FAQ)
   ↓
   Oferta irresistível + Remoção de objeções
   
5. CONVERSÃO
   ↓
   [Garantir meu acesso] → Checkout
```

## 📊 Hierarquia Visual

```
Nível 1 (Mais Importante)
├─ Hero Headline
├─ Oferta (Preço)
└─ CTAs Primários

Nível 2 (Importante)
├─ Section Titles
├─ Pain Points
└─ Prova Social

Nível 3 (Suporte)
├─ Descrições
├─ Benefícios
└─ FAQ

Nível 4 (Contexto)
├─ Footer
└─ Textos Legais
```

## 🎯 Pontos de Conversão

```
1. Hero CTA (Primary)
   └─ "Começar agora" → #oferta

2. Hero CTA (Secondary)
   └─ "Como funciona" → #como-funciona

3. Offer CTA (Main)
   └─ "Garantir meu acesso" → Checkout

4. Sticky CTA (Mobile)
   └─ "Garantir meu acesso - R$ 497" → #oferta

5. FAQ → Oferta
   └─ Após remover objeções, usuário volta à oferta
```

## 📏 Espaçamentos

```
Seções:
├─ Padding vertical: 64-96px (var(--spacing-2xl))
├─ Margin entre elementos: 32-48px (var(--spacing-lg))
└─ Gap em grids: 24-32px (var(--spacing-md))

Componentes:
├─ Padding interno: 24-32px (var(--spacing-md/lg))
├─ Margin bottom: 16px (var(--spacing-sm))
└─ Gap entre itens: 8-16px (var(--spacing-xs/sm))
```
