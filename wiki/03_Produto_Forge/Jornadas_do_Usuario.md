# Jornadas do Usuário (User Journeys)

## 🧠 Contexto BMAD
As jornadas de usuário definem os trilhos para o desenvolvimento de funcionalidades. No BMAD, cada jornada deve levar a um breakthrough (ex: conseguir uma entrevista). Este documento mapeia como o usuário "sobe de nível" no mundo real usando as ferramentas virtuais da Olcan.

## Conteúdo

### Perfil 1: O Aspirante Retirante
- **Início**: Descoberta via marketing -> Quiz Olcan Compass Core.
- **Engajamento**: Criação do companheiro Aura -> Exploração do Route OS.
- **Valor**: Uso do Narrative Forge para polir currículo -> Agendamento de mentoria.
- **Sucesso**: Conquista de contrato internacional -> Feedback na comunidade.

### Perfil 2: O Mentor Especialista
- **Início**: Cadastro no Marketplace -> Verificação de credenciais.
- **Operação**: Disponibilização de agenda -> Sessões de mentoria.
- **Sucesso**: Recebimento de pagamentos via Stripe -> Aumento de reputação no ecossistema.

### Pontos de Fricção Identificados
- Barreira de entrada no quiz (resolvido com UI Liquid Glass).
- Complexidade na configuração de pagamentos (Stripe Connect).
